# Crop Disease Identification System

## Overview
A Java-based intelligent system that uses machine learning to identify common crop diseases from leaf images. This web application helps farmers with early disease detection to prevent crop loss and improve yield.

## Features
- **Image Upload**: Simple interface for uploading plant leaf images
- **Disease Detection**: Identifies common plant diseases
- **Confidence Scoring**: Shows probability for each potential disease
- **Treatment Advice**: Provides actionable treatment recommendations
- **Multiple Crop Support**: Works with Tomato, Potato, and Corn plants
- **Web Interface**: User-friendly browser-based application

## Technologies/Tools Used
- **Backend**: Java, Spring Boot
- **Frontend**: HTML, CSS, JavaScript
- **Build Tool**: Maven
- **Machine Learning**: Deeplearning4j (Ready for integration)
- **Image Processing**: OpenCV Java bindings
